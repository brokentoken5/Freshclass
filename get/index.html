Get Freshclass

Follow our step-by-step guide on how to install Freshclass onto your site with one line of code.

# Step 1: Include the required tags

Before you add Freshclass to your site, it's highly recommended to add this tag before your HTML page to render elements consistently on less advanced browsers: 

<!DOCTYPE html>

and this <meta> tag so that your site works on mobile and turns it into a truly responsive one: 

<meta name="viewport" content="width=device-width, initial-scale=1">

# Step 2: Get the code

There are two ways to include our script. The first way is to include our stylesheet between the <head> tags of your page by pasting the following line of code: 

<link
  rel="stylesheet"
  href="https://freshclass.netlify.app/css/freshclass.min.css"
/>

The second way is to download our code, upload it to the root section of your page, and include it in the same format above. Don't worry, our code is open-source, so you can modify and share it any way you want.

⚠️ Warning

By downloading the freshclass source will not automatically update your script for new versions and bug fixes. It is recremonded to only download the source if you want the current version and dont want any updates.

# Step 3: Read the docs

To understand how to use Freshclass on your site, read the Documentation. It has simple information about the features Freshclass has and can help you make even more stylish sites, so don't forget to read the docs!
