Welcome to the officals Freshclass documentation, filled with information, guides, and examples on how to use Freshclass on your site. Knowing how to use Freshclass properly is critical when you create a site with it. Unlike many other classless CSS frameworks, Freshclass has many options, including:

  • Text
  • Containers
  • Quotes
  • Navigation bars
  • Side Bars
  • Buttons
  • Colored lists
  • Headers
  • Footers
  • Tables
  • Dropdowns
  • Code
  • And more...

This may seem like many confusing options, and if that's the case, reading our docs may help. Each of these features have articles with examples, information, and guides. So make sure to check out the guides and you'll be great at using Freshclass in no time!
